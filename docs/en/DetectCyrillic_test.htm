Detects coding and language - Russian or Ukrainian

Enter a piece of text in Russian or Ukrainian here and push the button Detect.
Tokens Count - number of tokens (words) after which the program starts analysis.
Detect All Languages - if not checked, the program detects Russian only.

Remarks. Do not use the additional script for coding switching in the right part of the menu - change the coding in your browser only. Older Netscape versions don't permit to change coding dynamically, if the document contains tag meta with the charset explicit definition.
